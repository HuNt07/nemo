
# Nemo

This is a Web Assembly port for the project
[emo](https://github.com/sokumon/emo)
\
In this port we use Natural Language instead of keyboard and mouse



## Build from source
Clone directory
```bash
   git clone https://github.com/hunt07/nemo
```
Change directory
```bash
   cd nemo
```
Install
```bash
   .\install_compiler.sh
```
Build again
```bash
   make clean
   make
```
Node install
```bash
   npm install
```
Server is starting
```bash
   node server.mjs
```
